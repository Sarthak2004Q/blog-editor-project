# Blog Editor Project

## Description
A full-stack blog editor with auto-save draft feature, built with Node.js, Express, MongoDB, and vanilla frontend (HTML, CSS, JS).

## Tech Stack
- Backend: Node.js, Express, MongoDB (Mongoose)
- Frontend: HTML, CSS, JavaScript

## How to Run

1. Clone or unzip the project.
2. Install backend dependencies:
3. Start MongoDB server:
- Run `mongod` or start MongoDB service.
4. Start backend server:
5. Open browser and visit:
http://localhost:5000


## Features
- Write, save drafts, publish blogs.
- Auto-save drafts every 5 seconds of inactivity.
- View list of published blogs and drafts.
- Edit existing drafts.
